About ST-Analyzer

ST-Analyzer is a web GUI, CLI, and API for quickly running common analysis, job management, and visualizing results.

To get started, create your project in the Projects menu, then schedule analysis to run in the Analysis menu.

Don't see the analysis you want? Consider creating or installing your own with the Plugins menu. Refer to the Documentation for ST-Analyzer concepts.
